# Page Actions

Every documentation page has two buttons in the top-right corner.

## Copy Markdown

Copies the current page content to your clipboard. Paste it into any AI chat,
note app, or editor.

## Open In...

Opens the current page directly in an AI tool:

- **ChatGPT** — Opens ChatGPT with this page ready to discuss
- **Claude** — Opens Claude with this page ready to discuss
- **T3 Chat** — Opens T3 Chat with this page ready to discuss
- **GitHub** — View the source file on GitHub
