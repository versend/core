# Versend

Un relay de webhooks ligero que envía notificaciones de despliegue de Vercel donde quieras. Despliégalo en minutos.

## Características

- **Multi-proveedor** — Envía a una plataforma o a varias simultáneamente
- **Tiempo real** — Las notificaciones llegan en segundos
- **Todos los eventos** — Despliegues, dominios, proyectos y más
- **Formato rico** — Formato nativo para cada plataforma
- **Contexto completo** — Rama, commit, entorno y enlaces
- **Confiable** — Reintentos automáticos con backoff para límites de tasa
- **Simple** — Despliega una vez, añade variables de entorno, listo
