# Documentación Amigable para IA

Esta documentación está diseñada para funcionar perfectamente con asistentes de IA. Puedes
pasar cualquier página (o todas) directamente a tu herramienta de IA favorita.

## Opciones Rápidas

| Quieres... | Haz esto |
|------------|----------|
| Preguntar a la IA sobre una página específica | Usa el menú **Abrir en** en cualquier página |
| Copiar una página para pegar | Haz clic en **Copiar Markdown** en cualquier página |
| Pasar toda la documentación a una IA | Usa `/llms-full.txt` |
| Obtener una sola página como texto | Añade `.mdx` a cualquier URL |
