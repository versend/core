# Versend

A lightweight webhook relay that sends Vercel deployment notifications wherever you want. Self-host it in minutes.

## Features

- **Multi-provider** — Send to one platform or many simultaneously
- **Real-time** — Notifications arrive within seconds
- **All events** — Deployments, domains, projects, and more
- **Rich formatting** — Native formatting for each platform
- **Context-rich** — Branch, commit, environment, and links
- **Reliable** — Automatic retry with backoff for rate limits
- **Simple** — Deploy once, add env vars, done
