# AI-Friendly Documentation

This documentation is designed to work seamlessly with AI assistants. You can
feed any page (or all pages) directly to your favorite AI tool.

## Quick Options

| Want to... | Do this |
|------------|---------|
| Ask AI about a specific page | Use the **Open in** dropdown on any page |
| Copy a page for pasting | Click **Copy Markdown** on any page |
| Feed all docs to an AI | Use `/llms-full.txt` |
| Get a single page as text | Add `.mdx` to any URL |
